# 📚 Aulas de JavaScript

Este repositório contém materiais, exemplos e exercícios desenvolvidos durante as aulas de **JavaScript**.  
O objetivo é servir como apoio ao aprendizado, reunindo códigos práticos e conceitos fundamentais da linguagem.

---

## 🚀 Conteúdo

- **Introdução ao JavaScript**
  - Variáveis, constantes e tipos de dados
  - Operadores básicos
- **Funções**
  - Declaração e chamada de funções
  - Funções anônimas e arrow functions
- **Estruturas de Controle**
  - Condicionais (`if`, `else`, `switch`)
  - Laços de repetição (`for`, `while`, `forEach`)
- **Manipulação de Arrays e Objetos**
  - Métodos úteis (`map`, `filter`, `reduce`)
- **DOM e Eventos**
  - Seleção de elementos
  - Manipulação de conteúdo e estilos
  - Eventos de clique, teclado e formulário
- **Projetos Práticos**
  - Exercícios e pequenos projetos para fixação

---

## 🛠️ Tecnologias Utilizadas

- [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)
- [Node.js](https://nodejs.org/) (quando necessário)
- HTML e CSS para exemplos de integração
